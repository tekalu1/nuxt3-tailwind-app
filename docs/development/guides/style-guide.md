# スタイルガイド

## 概要

本ドキュメントは、プロジェクト全体で一貫したUIスタイルを維持するためのガイドラインです。
Tailwind CSSのユーティリティクラスを使用する際のルールを定義しています。

## テキスト色

### グレースケール

| 用途 | クラス | 説明 |
|------|--------|------|
| メインテキスト | `text-gray-900` | 本文、見出し、重要な情報 |
| ラベル | `text-gray-700` | フォームラベル、説明文 |
| 補足テキスト | `text-gray-500` | 注釈、補助情報、プレースホルダー |
| 薄いテキスト | `text-gray-400` | 無効状態、セパレーター、装飾 |

### ステータス色

| 用途 | クラス | 説明 |
|------|--------|------|
| エラー | `text-red-500` | エラーメッセージ、失敗状態 |
| 成功 | `text-green-500` | 成功メッセージ、完了状態 |
| 警告 | `text-amber-500` | 警告メッセージ、注意喚起 |
| 情報 | `text-blue-500` | 情報メッセージ、ヒント |

### 使用例

```vue
<!-- メインテキスト -->
<h1 class="text-gray-900">タイトル</h1>

<!-- ラベル -->
<label class="text-gray-700">名前</label>

<!-- 補足テキスト -->
<p class="text-gray-500">※ 任意項目です</p>

<!-- エラーメッセージ -->
<span class="text-red-500">入力が必要です</span>
```

## ホバー遷移

ホバー状態での色遷移は**2段階以内**を原則とします。

### 推奨パターン

```
text-gray-500 → hover:text-gray-700  ✓ (2段階)
text-gray-400 → hover:text-gray-600  ✓ (2段階)
text-gray-500 → hover:text-gray-900  ✗ (4段階、避ける)
```

### 使用例

```vue
<!-- 正しい例 -->
<button class="text-gray-500 hover:text-gray-700">クリック</button>

<!-- 避けるべき例 -->
<button class="text-gray-500 hover:text-gray-900">クリック</button>
```

## フォントサイズ

### 標準クラスのみを使用

任意値（`text-[10px]`など）は使用せず、Tailwindの標準クラスを使用してください。

| クラス | サイズ | 用途 |
|--------|--------|------|
| `text-xs` | 12px | 補助テキスト、バッジ、タグ |
| `text-sm` | 14px | 本文、ボタン、入力欄 |
| `text-base` | 16px | 大きめの本文、見出し |
| `text-lg` | 18px | セクション見出し |
| `text-xl` | 20px | ページタイトル |

### 避けるべきパターン

```vue
<!-- 避けるべき例 -->
<span class="text-[10px]">小さいテキスト</span>
<span class="text-[13px]">任意サイズ</span>

<!-- 推奨例 -->
<span class="text-xs">小さいテキスト</span>
<span class="text-sm">本文テキスト</span>
```

## フォントファミリー

### デフォルトフォントの継承

フォントファミリーは親要素から継承されるため、明示的な指定は不要です。

```vue
<!-- 不要な指定 -->
<div class="font-sans">...</div>

<!-- 推奨：指定なし（親から継承） -->
<div>...</div>
```

### 例外：等幅フォント

コードやJSONの表示には`font-mono`を使用します。

```vue
<pre class="font-mono">{ "key": "value" }</pre>
<code class="font-mono">console.log()</code>
```

## アクセシビリティ

### コントラスト比

- メインテキスト（`text-gray-900`）：背景とのコントラスト比 4.5:1 以上
- 補足テキスト（`text-gray-500`）：背景とのコントラスト比 3:1 以上
- `text-gray-300`以下は装飾目的のみに使用し、重要な情報には使用しない

### フォーカス状態

フォーカス可能な要素には必ずフォーカスリングを表示します。

```vue
<button class="focus:ring-2 focus:ring-blue-500 focus:ring-offset-2">
  ボタン
</button>
```
